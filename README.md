## Introduction

This Java program provides a basic Employee Record Management System. It allows you to perform the following operations:

- Insert employee records
- Display employee records
- Search for employees by employee number
- Delete employee records
- Update employee records
- Sort employee records by employee number, employee name, or salary

The program uses Object Serialization to save and load employee records from a file.

## Features

- Simple command-line interface for managing employee records.
- Data is saved and loaded from a file using Java Object Serialization.
- Support for basic operations like insertion, display, search, deletion, and sorting.

## Prerequisites

- Java Development Kit (JDK) installed on your system.
- A Java IDE or a text editor for running and modifying the program.
